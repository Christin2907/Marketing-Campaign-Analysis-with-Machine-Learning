# Marketing Campaign Analysis with Machine Learning

## Project Overview

This project analyzes customer responses to marketing campaigns using
machine learning and statistical modeling.

The goal is to identify:

- campaign typologies
- customer response patterns
- drivers of campaign success
- high-response customer segments

The analysis combines clustering, sequence analysis, Markov chains and predictive models.

---

## Methods

The following analytical methods were applied:

- Feature Engineering & Data Preparation
- Campaign Typology (K-Means Clustering)
- Customer Response Typology (Sequence Analysis)
- Markov Chains (Customer Reaction Dynamics)
- Logistic Regression
- Random Forest Classification
- Feature Importance Analysis

---

## Dataset

The dataset contains customer demographic information, purchasing behavior and responses to five marketing campaigns.

Key variables include:

- Age
- Income
- Customer tenure
- Purchase channel activity
- Campaign responses

---

## Key Insights

- Web-active customers show significantly higher campaign response rates
- Premium campaigns perform better among high-income customers
- Long-term customers show more stable but slower reactions
- Response behavior follows identifiable patterns (Markov dynamics)
